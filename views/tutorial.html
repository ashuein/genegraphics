Finding and Loading Files

SEED Files

TSV files from the SEED website can be loaded into the Gene Graphics tool. In order to obtain a file from SEED, visit http://open.theseed.org/ and search for a gene in the center of the region you would like to display. For this example, we will use a gene found in Saccharomyces cerevisiae.

On the gene annotation page, scroll down to the tabbed box that shows the "Visual Region Information". Click on the tab that says "Tabular Region Information". Click on the "export table" button. You will be able to save a TSV file to your computer.

Once you have saved the file, navigate to the Gene Graphics tool. Click on "Input", then "Choose a File", Open the file that you saved from SEED.

The graphic should look similar to the image on the right. This file can also be pre-loaded into the Gene Graphics app by clicking Input > Tutorial Files > file1.tsv .
