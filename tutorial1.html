In this tutorial, we will use Gene Graphics to reproduce a figure similar to one that has been published.

DUF208, also identified as QueH has been identified as involved in the biosynthesis of the tRNA modification Queuosine. It is often associated with known queuosine related genes.

The figure to recreate can be found in the publication Zallot, et al., 2017, as Figure 1C and can be accessed at http://pubs.acs.org/doi/full/10.1021/acschembio.6b01100.

Identification of a Novel Epoxyqueuosine Reductase Family by Comparative Genomics Rémi Zallot, Robert Ross, Wei-Hung Chen, Steven D. Bruner, Patrick A. Limbach, and Valérie de Crécy-Lagard ACS Chemical Biology 2017 12 (3), 844-851 DOI: 10.1021/acschembio.6b01100

The image below is from the paper.

Fetching Data

The image is comprised of 5 genomes. Each genome can be fetched from NCBI using Input -> Fetch from NCBI.

The complete genome of Cryptobacterium curtum DSM 15641 is present in the NCBI nucleotide database as a genbank file under the entry CP001682.1. Selecting Genome Location, we can isolate the region between 1429479 and 1436151 to insert into the image the genes from TGT to QueF. You can enter ‘15641’ or ‘CP001682’ to find the entry.

Next we need to add genes from the genome of Syntrophomonas wolfei subsp. wolfei str. Goettingen G311. We will use the protein ID of TGT, as an entry point with the default window of 5000 bp. The unique protein identifier for TGT in that organism is ABI68735.1. This centers the view on TGT. Search by protein ID, with the default window size.

The next genome to add is centered on DUF208, from Eggerthella lenta DSM 2243. Search by gene symbol and organism. Use ‘duf208’ for Gene symbol and ‘Eggerthella’ to bring up ‘Eggerthella lenta DSM 2243’. The default window size is large enough to display the information needed.

To add the information for Solobacterium moorei DSM 22971, do a protein search with ID ‘EFW24078.1’ which cooresponds to DUF208 and a window size of 5000 bp.

The final genome section to fetch is from Atopobium rimae ATCC 49626. A protein search with EEE17942.1 allows us to display the genome neighborod of DUF208 in this genome.

Trimming

Now that the data is imported, we can edit the image to fit our design. For our first genome, from left to right, we have: TGT, HAD, DUF208(QueH), QueA, COG1738(YhhQ), QueC, and QueF. The initial information fetched shows the gene IDs present in the NCBI database.

Click on a gene to edit it. The gene currently being edited is highlighted in blue. You can adjust the font, background and text color, and alignment of the individual gene’s label using this editor.

Edit the name of each gene from left to right:

TGT, HAD, DUF208, QueA, COG1738, QueC, and QueF.

The colors of each gene can be changed to be similar to the published document, with blue, grey, yellow, green, black, red and purple, from left to right, respectively.

Note that to be seen, the Gene name for COG1738 should be colored white to show on a dark background.

For the second Genome, more information than needed is shown. TGT, the entry point, is the center gene. You can identify TGT by mousing over the light green arrow and noting the label as queuine tRNA-ribosyltransferase in the upper right corner.

Let’s remove the information that is not needed. To remove a gene, click on it, and then click the trash can image. It will ask you to confirm. The deleted gene will be removed, and the other genes will be moved to the left. Try removing Swol_1425.

We need to remove a number of genes:

  • From Synthrophomonas wolfei subsp. Wolfei str: Swol_1246, Swol_1427 and Swol_1431
  • From Eggertherlla lenta DSM 2243: Elen_0978, Elen_0979
  • From Eggertherlla lenta DSM 2243: Elen_0978, Elen_0979
  • From Solobacterium moorei F0204: HMPREF9430_01455, HMPREF9430_01456, HMPREF9430_01457, HMPREF9430_01458
  • From Atopobium rimae ATCC 49626: ATORI0001_0305, ATORI0001_0306, ATORI0001_0309

Modifying

In the original figure, the genes are on the same line. To adjust this parameter, go to Graph Settings -> Lane Settings, and unselect the first option so that all genes will be on the same line. This setting is on by default.

Now that we have all of our genes in place, we can edit them to match the original image. Selecting a gene that has been edited properly, DUF208 in the first genome, select the copy function icon and then select all other genes that will be labeled DUF208. Genes that are a destination for the selection are highlighted in red. Ensure that the option to apply the text from the original gene is also selected. When all desired genes are selected, click Paste to apply the settings.

Continue to do this for other genes, using the first genome as a template, for TGT, QueA and HAD.

To perform some final touch-ups, use Label Settings -> Gene Labels to move the gene names to be displayed under their corresponding arrow and center them on the arrow. COG1738 needs to be colored in black to become visible again. Label Settings -> Genome Labels will allow you to adjust the font to be bold and italic.

Finally, select each DUF208 gene to bold the text for that gene. This must be done individually.

When you are satisfied with the image, use Export to download a SVG or PNG of the image.
