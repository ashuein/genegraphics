The goal of this tutorial is to demonstrate how to fetch data from the NCBI website using the tool and how to manipulate the information to produce an image that matches your design requirements. You will produce an image similar to a figure that has been published.

DUF208, recently renamed QueH, has been identified as involved in the biosynthesis of the tRNA modification Queuosine. It is often associated with known queuosine related genes.

The figure to recreate can be found as Figure 1C in the publication Zallot, et al., 2017. It can be accessed at http://pubs.acs.org/doi/full/10.1021/acschembio.6b01100.

Identification of a Novel Epoxyqueuosine Reductase Family by Comparative Genomics Rémi Zallot, Robert Ross, Wei-Hung Chen, Steven D. Bruner, Patrick A. Limbach, and Valérie de Crécy-Lagard ACS Chemical Biology 2017 12 (3), 844-851 DOI: 10.1021/acschembio.6b01100

The image below is from the paper.

Fetching Data

The figure shows the gene clustering of DUF208 with Queuosine related genes in 5 genomes. All of the information shown can be fetched from NCBI using the "Input" -> "Fetch from NCBI" submenu.

The complete genome of Cryptobacterium curtum DSM 15641 is present in the NCBI nucleotide database as a genbank file under the entry CP001682.1. Selecting Genome Location, we can isolate the region between 1429479 and 1436151 to insert into the image the genes from TGT to QueF. You can enter ‘15641’ or ‘CP001682’ to quickly find the corresponding genome entry.

Next, we need to add genes from the genome of Syntrophomonas wolfei subsp. wolfei str. Goettingen G311. We will do a "Search by Protein ID", with the default window size, with the unique protein identifier for TGT in that organism: ABI68735.1. This centers the view on TGT.

We will then retrieve the information from Eggerthella lenta DSM 2243. Search by gene symbol and organism. Use ‘duf208’ for Gene symbol and ‘Eggerthella’ to bring up ‘Eggerthella lenta DSM 2243’ genome. The default window size is large enough to display the information needed.

To add the information for Solobacterium moorei DSM 22971, do a "Search by protein ID" with ID ‘EFW24078.1’ which corresponds to DUF208. Let's keep the standard window size of 5000 bp.

The final genome to add is from Atopobium rimae ATCC 49626. A "Search by protein ID" with EEE17942.1 allows us to display the genome neighborhood of DUF208 for this genome.

Trimming

Now that the data is imported, we can edit the image to our desired design. On the first genome, Cryptobacterium curtum DSM 15641, we have, from left to right: TGT, HAD, DUF208(QueH), QueA, COG1738(YhhQ), QueC, and QueF. The information fetched shows the gene IDs present in the NCBI database. We will change the gene names so it is more obvious what they are.

Click on a gene to edit it. The gene that is currently edited is highlighted in blue. You can adjust the font, background and text color, and alignment of the individual gene’s label using this editor.

Edit the name of each gene from left to right:

TGT, HAD, DUF208, QueA, COG1738, QueC, and QueF.

The colors of each gene can be changed to be similar to the published document, with blue, grey, yellow, green, black, red and purple, from left to right, respectively.

Note that to be seen, the gene name for COG1738 should be colored white to show on a dark background.

For the second genome, Syntrophomonas wolfei subsp. wolfei str. Goettingen G311, more information than needed is shown. TGT, the entry point, is the gene in the center. You can identify TGT by mousing over the light green arrow and noting the label "queuine tRNA-ribosyltransferase" in the upper right corner.

Let’s remove the information that is not needed. To remove a gene, click on it, and then click the trash can schematic. It will ask you to confirm. The deleted gene will be removed, and the other genes will be moved to the left. Try removing Swol_1425.

Overall, we need to remove several genes:

  • From Synthrophomonas wolfei subsp. Wolfei str. Goettingen G311: Swol_1246, Swol_1427 and Swol_1431
  • From Eggertherlla lenta DSM 2243: Elen_0978, Elen_0979
  • From Solobacterium moorei F0204: HMPREF9430_01455, HMPREF9430_01456, HMPREF9430_01457, HMPREF9430_01458
  • From Atopobium rimae ATCC 49626: ATORI0001_0305, ATORI0001_0306, ATORI0001_0309

Modifying

In the original figure, the genes are on the same line. To adjust this parameter, go to the "Graph Settings -> Lane Settings" menu, and unselect the first option so that all genes will be on the same line.

Now that we have all of our genes in place, we can edit them to match the original image. Selecting a gene that has been edited properly, DUF208 in the first genome, select the copy function icon and then select all other genes that will be labeled DUF208. Genes that are a destination for the selection are highlighted in red. Ensure that the option to apply the text from the original gene is also selected. When all desired genes are selected, click Paste to apply the settings.

Continue to do this for other genes, using the first genome as a template, for TGT, QueA and HAD.

To perform some final touch-ups, use the "Label Settings -> Gene Labels" menu to move the gene names to be displayed under their corresponding arrow and center them on the arrow. COG1738 needs to be colored in black to become visible again. The menu "Label Settings -> Genome Labels" will allow you to adjust the font to be bold and italic for all genomes at the same time.

Finally, select each DUF208 gene to bold the text for these genes. This must be done individually.

When you are satisfied with the image, use Export to download a SVG or PNG of the image. You can also download a TSV file that contains all the edited information.
