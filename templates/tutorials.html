Tutorials

Tutorial 1: DUF208
With this example, learn to obtain needed genomic information using the “Fetch from NCBI” menu, and learn how to manipulate an image to your expected design. This tutorial is focused on DUF208, a gene discovered involved in Queuosine biosynthesis.

Tutorial 2: Genbank File
Learn how to obtain a genome on NCBI Genbank and use it to create an image. This tutorial focuses on the folate-cleaving side-activity of ketopantoate hydroxymethyltransferase (PanB).

Tutorial 3: DUF89 Subfamily I Neighborhoods
Learn to acquire and then edit a TSV file from the SEED database. Upload the file to Genegraphics, edit the image, and export the image. This tutorial illustrates that DUF89 is associated with purine metabolism genes.
